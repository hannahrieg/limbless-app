# limbless
Web app for NGS sample/library/project tracking

## Installation
    - `pip install -r requirements.txt`
    - `wget https://github.com/twbs/bootstrap/archive/v5.3.2.zip`
    - `unzip v5.3.2.zip`
    - `mv bootstrap-5.3.2/limbless/static/bootstrap`

## Run flask debug server
    - `docker compose up`
    - `flask run`
