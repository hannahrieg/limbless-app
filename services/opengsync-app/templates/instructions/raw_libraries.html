Columns
  • Sample Name:
    • Biological Sample, e.g. sample_KO, sample_WT, ...
  • Library Name:
    • Name of the library, e.g. sample_KO_GEX, sample_KO_CMO, etc...
  • Library type:
    {% for library_type in LibraryType.as_list() %} {{ library_type.abbreviation }} {% endfor %}
  • Sequencing Depth 10X: number of cells in thousands / Non-10X: number of reads in millions. Specify numbers only!
  • Genome Specifies the reference genome used in downstream:
    {% for genome_ref in GenomeRef.as_list() %} {{ genome_ref.display_name }} {% endfor %}
Important
  • 'Sample Name'-column denotes a biological sample or specimen that will be prepared into one or more libraries.
  • Multiple rows can have same value in 'Sample Name'-column if multiple libraries are created from a single biological sample.
  • White-spaces in 'Sample Name'-column will be replaced with '_' automatically. Also non-alphanumeric characters (except '_', '-', '.') will be filtered.
  • In edge cases, please contact us at bsf@cemm.at
