6.2. Feature Reference Input Form

How to?
  • Specify predefined kit from Feature Kits
  • OR
    1. Download template
    2. Fill in the template
    3. Submit the file below
    Modify Columns
    • Library Name The sample from annotation sheet that this feature kit belongs to, must mach one and only one value from 'Sample Name'-column from sample annotation sheet
    • Kit name of a predefined feature kit, one found in Feature Kits page
    • Feature Name of the feature.
    • Sequence e.g. 'TGAGTACAGGCTCCCAAG'
    • Pattern e.g. '^NNNNNNNNNNNNNNNNNNNNNN(BC)'
    • Read e.g. 'R2'
    Important
    • Three ways to specify Features:
      1. If using the same predefined kit from Feature Kits for all Antibody Capture Libraries, select kit from the search bar below.
      2. If using predefined kit from Feature Kits: specify columns Sample Name, Feature Kit
      3. If using custom kit: specify columns Sample Name, Feature, Sequence, Pattern, Read
    • If a custom feature is part of all antibody libraries, leave 'Sample Name'-column empty.
