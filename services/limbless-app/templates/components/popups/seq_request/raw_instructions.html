How to?
  • Fill in the required columns and submit as a file or live spreadsheet.
Required Columns
  • Sample Name Biological Sample, e.g. sample_KO, sample_WT, etc...
  • Library Name Name of the library, e.g. sample_KO_GEX, sample_KO_CMO, etc...
  • Library type:
      {% for _, library_type in LibraryType.as_tuples() %}
    • {{ library_type.description }}
      • {% endfor %}
Optional Columns (can be left blank)
  • Organism Must be specified if samples from two or more organisms are present.
      {% for _, organism in Organism.as_tuples() %}
    • {{ organism.name }}
      • {% endfor %}
  • Project Must be specified if request contains samples from two or more projects.
Important
  • 'Sample Name'-column denotes a biological sample or specimen that will be prepared into one or more libraries.
  • Multiple rows can have same value in 'Sample Name'-column if multiple libraries are created from a single biological sample.
  • White-spaces in 'Sample Name'-column will be replaced with '_' automatically. Also non-alphanumeric characters (except '_', '-', '.') will be filtered.
  • In edge cases, please contact us at bsf@cemm.at
